# DeepSeek Usage Enhancer

> Forked from [Jmkwang/DeepSeek-Usage-Enhancer](https://github.com/Jmkwang/DeepSeek-Usage-Enhancer)

在 DeepSeek 用量页面自动补全今日消费、请求次数、Token 用量、缓存命中率。

## 版本

| 版本 | 文件 | 特点 |
|------|------|------|
| **页面注入版** | `DeepSeek-Usage-Enhancer.js` | 数据注入页面布局，像原生功能 |
| **悬浮面板版** | `deepseek-usage-monitor.js` | 独立浮窗，可拖拽折叠 |
| **悬浮面板版（中文）** | `deepseek-usage-monitor-zh.js` | 独立浮窗，全中文界面，日历月/周/范围三种视图 & 主题切换 |

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 新建脚本，粘贴对应 `.js` 文件内容，保存
3. 打开 [DeepSeek 用量页面](https://platform.deepseek.com/usage) 即可

## 功能特色（中文版独有）

### 📅 历史用量日历（三视图）
点击面板右上角 `📅` 按钮打开日历，支持三种查看模式：

- **月视图** — 传统日历网格，每个日期显示当日费用，悬停查看 Token/请求/命中率详情，底部显示月度合计（含 Pro/Flash Token 细分）
- **周视图** — 按自然周聚合，每行显示周标签、费用、Token、请求、命中率，悬停查看 Pro/Flash 详细拆分，跨月自动加载
- **范围视图** — 点击起始日期和结束日期，中间日期高亮显示，底部显示范围合计统计

> 月份导航自动同步 DeepSeek 页面原生月份选择器。在范围选择模式下可通过 ◀/▶ 切换月份以选取不同日期的起止范围。

### 🌓 深色/浅色主题
点击 `🌓` 按钮切换主题，偏好自动保存。暗色主题采用 Cyber Teal 风格，浅色主题为冷灰系。

### 📊 完整数据面板
- **账户余额** — 充值余额、本月消费、今日消费
- **各模型详情** — deepseek-v4-pro 和 deepseek-v4-flash 的请求数、总 Token、缓存命中/未命中/输出 Token、缓存命中率
- **状态栏** — 汇总总 Token、总请求数、总体缓存命中率

### 🔧 交互功能
- **拖拽移动** — 拖拽标题栏移动面板，位置自动保存
- **折叠/展开** — 点击 ▼ 收起面板
- **数值切换** — 点击任意数字在短格式（1.5K）和原始值（1,500）之间切换
- **返回今日** — 查看历史数据后一键回到今日视图
- **键盘导航** — 日历中 ←/→ 切换月份，Enter 查看详情，Escape 关闭

## 原理

拦截页面 API 请求（`window.fetch` + `XMLHttpRequest`），按月份缓存数据，在浏览器本地处理，不向第三方发送。

## 安全

- 纯本地运行，`@grant none`，无远程通信
- 源代码可逐行审查
